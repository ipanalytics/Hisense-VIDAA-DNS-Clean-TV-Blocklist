# Hisense-VIDAA-DNS-Clean-TV-Blocklist
Private DNS blocklist for a Hisense Smart TV running VIDAA OS in Europe / Germany.  This list is made for a TV used mostly as a clean Smart TV screen for apps like YouTube and Netflix, without German live TV, HbbTV, Red Button, VIDAA Free channels, recommendations, promo tiles, banners, and other home-screen clutter.
